v.0.1.5¶

Improvements:¶

  1. Possibility to update version with versioneer.py script and version parameter is added (no package installation is required).
  2. What_is_new page is added for version change control.
  3. Debug logging is extended (blocks, erroneous requests).
  4. Code comments are improved.
  5. Default configs are changes, shortened config is added.

Bugs fixed:¶

  1. Paths to sources are fixed in .rst files.
  2. Platform check is deleted.
  3. Waste query parameters are fixed in /search.
  4. Bug about wrong order of persons and list blocking is fixed.
  5. Bug about person removed from list is fixed.
