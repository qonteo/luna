patch(token_id)[source]¶

Resource to change token data

PATCH /account/tokens/{token_id}¶
Parameters:
  • token_id – token id

Example request:

Request Headers:
 
  • Authorization – basic authorization
  • LUNA-Request-Id – request id
Status Codes:
  • 204 No Content – token is changed successfully
 
POST /storage/tokens/{token_id} HTTP/1.1
Accept: application/json
LUNA-Request-Id: 1516179740,c06887a2

Token data must be presented in format token_data.

Error message is returned in format server_error.

Status Codes:
  • 400 Bad Request – field token_data is too large
  • 400 Bad Request – field token_data has wrong type, string type is required
  • 400 Bad Request – descriptor not found
  • 500 Internal Server Error – internal server error
