Tokens API¶

class luna_api.app.rest_handlers.tokens_handler.TokensHandler(application, request, **kwargs)[source]¶

Handler to operate with account tokens. To operate with them, you should authorize into account and account must be active.

delete()[source]¶

Request to delete the list of account tokens

DELETE /account/tokens¶

Example request:

Request Headers:
 
  • Authorization – basic authorization
  • LUNA-Request-Id – request id
 
POST /account/tokens HTTP/1.1
Accept: application/json
json to delete tokens¶
Object Properties:
 
  • tokens (_list_(uuid4)) – tokens for deletion
JSON Example
{
    "tokens": [
        "3549816c-e95e-4fdb-a70f-56c872e3c3bf",
        "000188a8-7c64-4a35-8149-0a8716857baf",
        "2807e5b9-fd21-40e8-960c-f14f8bed7373"
    ]
}

Example response:

HTTP/1.1 204 Ok
Vary: Accept
LUNA-Request-Id: 1516179740,c06887a2

Error message is returned in format server_error.

Status Codes:
  • 400 Bad Request – no json in request
  • 400 Bad Request – no tokens field in json
  • 400 Bad Request – tokens field is not a list
  • 500 Internal Server Error – internal server error
get()[source]¶

Resource to get all account tokens

GET /account/tokens¶
Optparam page:

A number of page. Minimum 1, default 1.
Optparam page_size:
 

Number of tokens of the same type on page. Minimum 1, maximum 100, default 10.
Request Headers:
 
  • Authorization – basic authorization
  • LUNA-Request-Id – request id

Example response:

HTTP/1.1 200 Ok
Vary: Accept
Content-Type: application/json
LUNA-Request-Id: 1516179740,c06887a2
tokens¶
Object Properties:
 
  • tokens (_list_(token_data)) – token list with token_data
  • count (int) – number of tokens
JSON Example
{
    "tokens": [
        {
            "token_data": "900 Rachel Fields Suite 769"
        },
        {
            "token_data": "7927 Montoya Keys Suite 859"
        },
        {
            "token_data": "915 Carter Islands"
        }
    ],
    "count": 1430
}

Error message is returned in format server_error.

Status Codes:
  • 500 Internal Server Error – internal server error
post()[source]¶

Request for creation of new token for the account

POST /account/tokens¶
Request Headers:
 
  • Authorization – basic authorization
  • LUNA-Request-Id – request id
 
POST /storage/tokens HTTP/1.1
Accept: application/json

Json to attach token data must be in format token_data.

Example response:

HTTP/1.1 201 Ok
Vary: Accept
Content-Type: application/json
LUNA-Request-Id: 1516179740,c06887a2
token¶
Object Properties:
 
  • token (uuid4) – new token id
JSON Example
{
    "token": "bba21399-602b-477b-8e29-01d9d54b26d9"
}

Error message is returned in format server_error.

Status Codes:
  • 400 Bad Request – field token_data is too large
  • 400 Bad Request – field token_data has wrong type, string field is required
  • 500 Internal Server Error – internal server error
