patch(person_id)[source]¶

Attach/detach descriptor to person.

PATCH /storage/persons/{person_id}/linked_descriptors?do=attach&descriptor_id=16fd2706-8baf-433b-82eb-8c7fada847da¶
Parameters:
  • person_id – person id
Optparam do:

‘attach’ or ‘detach’
Optparam descriptor_id:
 

descriptor id

Example request:

Request Headers:
 
  • Authorization – basic authorization

or

Request Headers:
 
  • X-Auth-Token – ‘16fd2706-8baf-433b-82eb-8c7fada847da’
  • LUNA-Request-Id – request id

Example response:

HTTP/1.1 204
Vary: Accept
LUNA-Request-Id: 1516179740,c06887a2
Status Codes:
  • 204 No Content – Descriptor is successfully attached/detached

Error message is returned in format server_error.

Status Codes:
  • 400 Bad Request – bad query parameters do
  • 400 Bad Request – account list with corresponding type not found
  • 400 Bad Request – required parameters ‘descriptor_id’ not found
  • 400 Bad Request – object in query is not UUID4, format: ‘xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx’
  • 403 Forbidden – person is currently blocked
  • 404 Not Found – person is not found
  • 409 Conflict – person or descriptor are already in the list
  • 500 Internal Server Error – internal server error
