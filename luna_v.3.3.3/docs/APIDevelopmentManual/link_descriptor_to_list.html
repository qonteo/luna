Handler to add or delete descriptors in lists.

get(photo_id)[source]¶

Get all account lists, descriptor is linked to.

GET /storage/descriptors/{photo_id}/linked_lists¶
Parameters:
  • photo_id – descriptor id

Example request:

Request Headers:
 
  • LUNA-Request-Id – request id
  • Authorization – basic authorization

or

Request Headers:
 
  • X-Auth-Token – ‘16fd2706-8baf-433b-82eb-8c7fada847da’
  • LUNA-Request-Id – request id

Example response:

HTTP/1.1 200 Ok
Vary: Accept
Content-Type: application/json
LUNA-Request-Id: 1516179740,c06887a2
Status Codes:
  • 200 OK – Lists received successfully
descriptor_lists¶
Object Properties:
 
  • lists (_list_(uuid4)) – all lists

{
    "lists": [

Error message is returned in format server_error.

Status Codes:
  • 404 Not Found – descriptor not found
  • 500 Internal Server Error – internal server error
patch(photo_id)[source]¶

Attach/detach descriptor to account list.

PATCH /storage/descriptors/{photo_id}/linked_lists?do=attach&list_id=16fd2706-8baf-433b-82eb-8c7fada847da¶
Parameters:
  • photo_id – descriptor id
Optparam do:

‘attach’ or ‘detach’
Optparam list_id:
 

list id

Example request:

Request Headers:
 
  • Authorization – basic authorization

or

Request Headers:
 
  • X-Auth-Token – ‘16fd2706-8baf-433b-82eb-8c7fada847da’
  • LUNA-Request-Id – request id

Example response:

HTTP/1.1 204
LUNA-Request-Id: 1516179740,c06887a2
Status Codes:
  • 204 No Content – Descriptor is successfully attached/detached

Error message is returned in format server_error.

Status Codes:
  • 400 Bad Request – list not found
  • 400 Bad Request – Bad query parameters do
  • 400 Bad Request – Account list with corresponding type not found
  • 400 Bad Request – Object in query is not UUID4, format: xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx’
  • 403 Forbidden – list or person is blocked
  • 404 Not Found – descriptor not found
  • 409 Conflict – descriptor is already in list
  • 500 Internal Server Error – internal server error
