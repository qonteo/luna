v.1.0.0¶

Improvements:¶

  1. Added TestAsyncLink to main tests.
  2. Optimized work time of tests.
  3. Added S3_OFF parameters in /test/config.py for skipping test which using S3.
  4. Changed default name db for statistic (admin_stats_db -> luna_api_admin).
  5. Updated information about limiting of influx in install.
  6. Added parameter DNS_SERVER to /configs/config.py, this parameter is used for determinate ip address
  7. Added print about success checking connection to postgres
  8. Added support images and descriptors in base64 encoding. For using see /storage/descriptors in raml.
  9. Added upload scripts, see Upload scripts.
  10. Add resource “/version” to raml.
  11. Return version of Luna Core in “/version”.
  12. Refactoring docstrings.
  13. Add CORS for resource “/version”.
  14. Added description about connection to extarnal hosts, see Connect to external hosts.
  15. Auto generate api version for raml.
  16. Support Upper letters in email.
  17. Added errors in cases when LUNA core unavailable.
  18. Added errors in cases when Image is corrupted (Error.ConvertImageToJPGError).

Bugs fixed:¶

  1. Fixed TestAsyncLink check correct status code now
  2. Exceptions are thrown in the function to obtain an IP address without the Internet.
  3. Fixed handler of error in saving portrait with plugins.
  4. Fixed trying to save portrait in plugin when extract_descriptor=1.

How to update from previous version:¶

To update from previous version simply replace folders app, error, base_scripts, tests, raml and file version.py. It is strongly recommended to backup DB postgres and server.
