person¶
Object Properties:
 
  • user_data (user_name) – person information
  • id (uuid4) – person id
  • descriptors (_list_(uuid4)) – list of descriptors, linked to person.
  • create_time (iso8601) – person creation time with timezone
  • lists (_list_(uuid4)) – list of accounts list ids, person is linked to.
JSON Example
{
    "user_data": "wvazquez",
    "id": "a19c6f71-2970-4464-8073-d2691eb5c6af",
    "descriptors": [
        "19d6e337-26e9-45c6-8739-54791ad2d4f6",
        "9fc5de80-6fcd-4be8-8b64-1db7f69d5381",
        "9bb2f6cf-9f58-4849-b6ad-213455bb067f"
    ],
    "create_time": "1983-10-09T13:32:37",
    "lists": [
        "174bed53-0c4c-4bf5-8815-81c74ea8cf2d",
        "37f8c8a9-bc55-41f9-b236-afed240d3f0f",
        "e7e40440-b591-4e99-8dec-aa7558aad0a0"
    ]
}
descriptor¶
Object Properties:
 
  • id (uuid4) – descriptor id
  • person_id (uuid4) – descriptor id, person is linked to (null, if descriptor is not linked to any person)
  • last_update (iso8601) – time of last descriptor attach/detach to lists, persons
  • lists (_list_(uuid4)) – list of account list ids descriptor is linked to
JSON Example
{
    "id": "7b10f282-54a7-483f-b99d-05334eee7244",
    "person_id": "d0488385-1c0e-4d7b-9a9e-f92a37360cf9",
    "last_update": "2014-07-22T05:52:11",
    "lists": [
        "70ad7ab8-822c-4b75-bc6a-78b5c9d25d89",
        "ffc30859-64e2-4e6f-b291-72d0d71149e9",
        "94d12599-239c-480e-a388-4dc4345aac2e"
    ]
}

qualityAttributes¶
Object Properties:
 
  • light (float) – Low value means overlit face area (i.e. overbright lighting, sensor overexposure), minimum: 0 - maximum: 1.
  • dark (float) – Low value means underlit face area (i.e. due to backlight, poor lighting, sensor underexposure), minimum: 0 - maximum: 1.
  • saturation (float) – Low value means low saturation (e.g. grayscale images), minimum: 0 - maximum: 1.
  • blurriness (float) – Low value means blurred image (e.g. due to depth of field or motion blur). High value means sharp image, minimum: 0 - maximum: 1.
JSON Example
{
    "light": 0.16024802999999999,
    "dark": 0.0,
    "saturation": 0.262939453125,
    "blurriness": 0.910003662109375
}
