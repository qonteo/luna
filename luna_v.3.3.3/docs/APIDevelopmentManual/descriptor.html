Descriptor API¶

class luna_api.app.rest_handlers.descriptor_handler.DescriptorHandler(application, request, **kwargs)[source]¶

Handler to get information about an existing descriptor. To work with this handler you have to log in and account must be active.

get(descriptor_id)[source]¶

Resource to get descriptor data

GET /storage/descriptors/{descriptor_id}¶
Parameters:
  • descriptor_id – descriptor id

Example request:

Request Headers:
 
  • Authorization – basic authorization

or

Request Headers:
 
  • X-Auth-Token – ‘16fd2706-8baf-433b-82eb-8c7fada847da’
  • LUNA-Request-Id – request id

Example response:

HTTP/1.1 200 Ok
Vary: Accept
Content-Type: application/json
LUNA-Request-Id: 1516179740,c06887a2
Status Codes:
  • 200 OK – Information about person is received successfully
descriptor information¶
Object Properties:
 
  • id (descriptor) – descriptor id
JSON Example
{
    "id": "{descriptor object}"
}

Error message is given in server_error format

Status Codes:
  • 404 Not Found – descriptor not found
  • 500 Internal Server Error – internal server error
