class luna_faces.app.handlers.faces_attributes_handler.FaceAttributesHandler(application, request, **kwargs)[source]¶

Handler for work with face attributes

get(faceId)[source]¶

Request to get the face attributes id.

Parameters:faceId (str) – face id

Resource is reached by address ‘/faces/{faceId}/attributes’

GET /faces/{faceId}/attributes¶
Parameters:
  • faceId – face id
Query Parameters:
 
  • account_id – account id, this parameter determine, that action must be done with only with objects of this account.
Request Headers:
 
  • LUNA-Request-Id – request id
 
GET /faces/{faceId}/attributes HTTP/1.1

Example response:

HTTP/1.1 200
LUNA-Request-Id: 1516179740,af3fb041-1a55-43e6-9ac2-a361aa43295f
Begin-Request-Time: 1526039272.9173293
End-Request-Time: 1526039272.9505265
Content-Type: application/json

Output face will be represent in luna_face_attributes.

Status Codes:
  • 200 OK – Ok
  • 404 Not Found – face not found
  • 500 Internal Server Error – internal server error
