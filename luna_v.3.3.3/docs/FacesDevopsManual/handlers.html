Common jsons¶

luna_face¶
Object Properties:
 
  • face_id (uuid4) – face id
  • event_id (uuid4) – event id
  • attributes_id (uuid4) – attribute id
  • create_time (iso8601) – time of face creating
  • user_data (user_name) – user data about face
  • account_id (uuid4) – account id
  • external_id (string) – external id
JSON Example
{
    "face_id": "2d15fab6-d943-4084-a02d-f3753f4832ec",
    "event_id": "2b4b6f4c-6936-4d54-8d0b-98c08a17c052",
    "attributes_id": "0e65bf9e-cc50-4361-8f70-bc0418f7a1ab",
    "create_time": "1972-11-28T17:48:52",
    "user_data": "kelly88",
    "account_id": "128765e9-41a3-4c10-9c95-99630916d43f",
    "external_id": "oaNfypJjxQcefGtLHuhS"
}

luna_list¶
Object Properties:
 
  • list_id (uuid4) – list id
  • account_id (uuid4) – account id
  • user_data (user_name) – user data about list
  • face_count (integer) – face count in list
  • create_time (iso8601) –
    • list create time
  • last_update_time (iso8601) – time of last action with list (attach or detach face from list)
luna_list_with_keys¶
Object Properties:
 
  • list_id (uuid4) – list id
  • link_key (integer) – link key
  • unlink_key (integer) – unlink key

log_note¶
Object Properties:
 
  • list_id (uuid4) – list id
  • attributes_id (uuid4) – attributes id
  • face_count (integer) – face count in list
  • action (_enum_(0,1)) – type of action
  • update_time (iso8601) – time of action (attach or detach attributes from list)
