List handler¶

class luna_faces.app.handlers.linker_handler.LinkerHandler(application, request, **kwargs)[source]¶

Handler for work with list

patch()[source]¶

Link or unlink face to/from list

Resource is reached by address ‘/linker’

PATCH /linker¶
Request Headers:
 
  • LUNA-Request-Id – request id
  • Content-Type – application/json
Query Parameters:
 
  • account_id – account id, this parameter determinate, that action must be done with only with objects of this account.
 
PATCH /linker HTTP/1.1
luna_linker¶
Object Properties:
 
  • action (attach or detach) – attach faces to list or detach
  • face_ids (_list_uuid4) – face ids
  • list_id (uuid4) – list id
JSON Example
{
    "action": "{attach or detach object}",
    "face_ids": [
        {
            "uid": "{uid object}"
        },
        {
            "uid": "{uid object}"
        },
        {
            "uid": "{uid object}"
        }
    ],
    "list_id": "bd6297d5-3e00-4aa5-8ae8-01543199dba7"
}

Example response:

HTTP/1.1 204
Begin-Request-Time: 1526039272.9173293
End-Request-Time: 1526039272.9505265
LUNA-Request-Id: 1516179740,af3fb041-1a55-43e6-9ac2-a361aa43295f
Status Codes:
  • 204 No Content – Ok
  • 400 Bad Request – list not found
  • 400 Bad Request – one or more faces not found
  • 500 Internal Server Error – internal server error
