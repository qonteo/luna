put(faceId)[source]¶

Resource is reached by address ‘ /faces/{faceId}’

PUT /faces/{faceId}¶

Request to create face.

Request Headers:
 
  • LUNA-Request-Id – request id
 
PUT  /faces/{faceId} HTTP/1.1
Accept: application/json
luna_put_face¶
Object Properties:
 
  • account_id (uuid4) – id of account, required
  • attributes_id (uuid4) – attributes id
  • user_data (user_name) – face information
  • event_id (uuid4) – reference to event which created face
  • external_id (str) – external id of the face, if it has its own mapping in external system
JSON Example
{
    "account_id": "99189e0e-bc48-49ab-9327-0394346ca88a",
    "attributes_id": "b7f466f4-37a9-4902-92c5-7979bb38800d",
    "user_data": "bowenmichelle",
    "event_id": "dce6a785-37ac-458f-a4e5-ca471f213b57",
    "external_id": "fdfmjKkmbyeTiZXKykMZ"
}

HTTP/1.1 200 Ok
Content-Type: application/json
Begin-Request-Time: 1526039272.9173293
End-Request-Time: 1526039272.9505265
LUNA-Request-Id: 1516179740,af3fb041-1a55-43e6-9ac2-a361aa43295f
Location: /faces/{face_id}
face_id¶
Object Properties:
 
  • face_id (uuid4) – id of created face
JSON Example
{
    "face_id": "83fbaaf8-5e01-42f8-ab16-3a592960f1d2"
}

Error message is returned on format server_error.

Status Codes:
  • 201 Created – face successfully create
  • 400 Bad Request – field user_data is too large
  • 400 Bad Request – field user_data has wrong type, string type is required
  • 409 Conflict – face with sane attribute_id already exist
  • 500 Internal Server Error – internal server error
