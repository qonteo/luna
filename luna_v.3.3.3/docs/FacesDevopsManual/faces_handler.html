delete()[source]¶

Delete faces

Resource is reached by address ‘/faces’

DELETE /faces¶
Request Headers:
 
  • LUNA-Request-Id – request id
Query Parameters:
 
  • account_id – account id, this parameter determinate, that action must be done with only with objects of this account.
 
DELETE /faces/ HTTP/1.1

get()[source]¶

Resource is reached by address ‘/faces’

GET /faces¶
Query Parameters:
 
  • page – page count, default 1
  • page_size – page size, default 10
  • user_data – user data
  • time__lt – upper bound of face create time
  • time__gte – lower bound of face create time
  • event_id – event id
  • list_id – list id
  • account_id – account id
  • face_ids – list of face ids
  • externalId – external id of the face, if it has its own mapping in external system
Request Headers:
 
  • LUNA-Request-Id – request id
 
GET /faces HTTP/1.1

Example response:

post()[source]¶

Resource is reached by address ‘/faces’

POST /faces¶

Request to create face.

Request Headers:
 
  • LUNA-Request-Id – request id
 
POST /faces HTTP/1.1
Accept: application/json
luna_create_face¶
Object Properties:
 
  • account_id (uuid4) – id of account, required
  • attributes_id (uuid4) – attributes id
  • user_data (user_name) – face information
  • event_id (uuid4) – reference to event which created face
  • externalId (str) – external id of the face, if it has its own mapping in external system
JSON Example
{
    "account_id": "99f19c0b-4eff-41ae-b554-21200035ab7d",
    "attributes_id": "e0742634-d7dd-4fe2-9876-0aabc29d0852",
    "user_data": "hunter13",
    "event_id": "65da5210-bc28-4305-bd68-547df629629a",
    "externalId": "ogSqdPZOEOXukStiYUBg"
}

HTTP/1.1 201 Created
Content-Type: application/json
Begin-Request-Time: 1526039272.9173293
End-Request-Time: 1526039272.9505265
LUNA-Request-Id: 1516179740,af3fb041-1a55-43e6-9ac2-a361aa43295f
Location: /faces/{face_id}
face_id
Object Properties:
 
  • face_id (uuid4) – id of created face
JSON Example
{
    "face_id": "d96e5660-2d77-4be5-aa4f-7c27ae7794f8"
}

Error message is returned on format server_error.

Status Codes:
  • 201 Created – face successfully create
  • 400 Bad Request – field user_data is too large
  • 400 Bad Request – field user_data has wrong type, string type is required
  • 409 Conflict – face with sane attribute_id already exist
  • 500 Internal Server Error – internal server error
