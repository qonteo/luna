delete()[source]¶

Delete list

Request to remove lists.

Resource is reached by address ‘/lists’

DELETE /lists¶
Request Headers:
 
  • LUNA-Request-Id – request id
Query Parameters:
 
  • account_id – account id, this parameter determinate, that action must be done with only with objects of this account.
 
DELETE /lists/ HTTP/1.1

Example response:

HTTP/1.1 204

  • 400 Bad Request – one or more lists not found
  • 500 Internal Server Error – internal server error
