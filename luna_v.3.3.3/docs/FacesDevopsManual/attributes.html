Handler for work with attributes

get()[source]¶

Resource is reached by address ‘/attributes’

GET /attributes¶
Query Parameters:
 
  • page – page count, default 1
  • page_size – page size, default 10
  • time__lt – upper bound of attributes last update time
  • time__gte – lower bound of face create time
  • list_id – list id
  • account_id – account id
Request Headers:
 
  • LUNA-Request-Id – request id
 
GET /attributes HTTP/1.1

Example response:

HTTP/1.1 200 OK
