v.1.2.6¶

Improvements:¶

  1. Added ability of bucket deletion.
  2. Added ability of cache for images.
  3. Added ability of work with text and json objects.
  4. Added check connection to database on startup
  5. Remove loggers for each file handler, make global logger
  6. Update tornado version (5.0.1).
  7. Now ORIGIN is used in configs. Origin consists of “<schema>://<ip-address>:<port>”
  8. Add method HEAD for /buckets/{bucketName}/images/{imageId}{thumbnail} and /buckets/{bucketName}/objects/{objectId}

Bugs fixed:¶

  1. Fix integration with S3-compitable storage.

How to update from previous version:¶

To update from previous version simply replace folders app, errors, image, preview, storage, tests, docs and file version.py.
