DELETE /buckets/{bucketName}/images¶

Delete images from bucket

Parameters:
  • bucketName – bucket name
Request Headers:
 
  • LUNA-Request-Id – request id

Example request:

DELETE /buckets/fd5a99ad-6ca3-4019-9bb0-910c0044f676/images HTTP/1.1
Content-Type: application/json

Body will be represented in format delete_images_response.

HTTP/1.1 204

POST /buckets/{bucketName}/images¶

Post image to bucket

Parameters:

Request Headers:
 
  • LUNA-Request-Id – request id

Example request:

POST /buckets/fd5a99ad-6ca3-4019-9bb0-910c0044f676/images HTTP/1.1
Content-Type: image/jpeg

Example response:

HTTP/1.1 201

Content-Type: application/json
