class luna_admin.app.handlers.account_handler.AccountHandler(application, request, **kwargs)[source]¶

Handler for work with an account.

get(account_id)[source]¶

Get account info.

GET /accounts/{account_id}¶
Parameters:
  • account_id – account id
Request Headers:
 
  • Authorization – basic authorization

Example response:

HTTP/1.1 200 Ok
Vary: Accept
Content-Type: application/json
LUNA-Request-Id: 1516179740,d3abc2f6-70f1-4ae0-9d10-475103e0891d

Output account will be represent in account

Message error is returned in format server_error.

Status Codes:
  • 200 OK – account was gotten successfully.
  • 404 Not Found – Account not found
  • 500 Internal Server Error – internal server error
Raises:VLException(Error.AccountNotFound, 404, isCriticalError=False) – if account not found
Return type:Generator[None, None, None]
