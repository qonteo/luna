Tokens handler¶

class luna_admin.app.handlers.account_tokens_handler.AccountTokensHandler(application, request, **kwargs)[source]¶

Handler of account tokens

get(account_id)[source]¶

Get tokens of account with pagination.

GET /accounts/{account_id}¶
Parameters:
  • account_id – account id
Query Parameters:
 
  • page – page count, default 1
  • page_size – page size, default 10
Request Headers:
 
  • Authorization – basic authorization

Example response:

HTTP/1.1 200 Ok
Vary: Accept
Content-Type: application/json
LUNA-Request-Id: 1516179740,d3abc2f6-70f1-4ae0-9d10-475103e0891d
list of account tokens¶
Object Properties:
 
  • tokens (_list_(auth_token)) – tokens
  • token_count (integer) – token count
JSON Example
{
    "tokens": [
        {
            "token_id": "2c26657b-c9ca-4e7d-b1f2-9fe6940a12ed",

Message error is returned in format server_error.

Status Codes:
  • 200 OK – success
  • 400 Bad Request – Bad query parameters
  • 500 Internal Server Error – internal server error
