Welcome to LUNA Administration panel service documentation!¶

Primary features of the service are:
  1. Viewing accounts of registered clients;
  2. Blocking and unblocking the clients;
  3. Launching garbage collectors non-linked faces;
  4. Viewing results of garbage collecting;
  5. Run re-extract descriptors.
