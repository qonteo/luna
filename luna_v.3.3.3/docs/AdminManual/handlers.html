Common jsons¶

server_error¶
Object Properties:
 
  • error_code (integer) – inner error code, for more info about errors go to Service errors.
  • detail (string) – error description
JSON Example
{
    "error_code": 763,
    "detail": "mSroErobYXYiKPsOcywK"
}
account_info¶
Object Properties:
 
  • account_id (uuid4) – account id
  • email (email) – email of account
  • organization_name (organization_name) – organization name
  • status (boolean) – if account is currently blocked, system returns True
JSON Example
{
    "account_id": "b8262b9f-b79d-461a-86c8-d04dafa270e0",
    "email": "cjones@mercer-spencer.com",
    "organization_name": "{organization_name object}",
    "status": true
}
account_stats¶
Object Properties:
 
  • person_count (integer) – person count of account
  • list_count (integer) – list count of account
  • descriptor_count (integer) – descriptor count of account
  • token_count (integer) – token count of account
JSON Example
{
    "person_count": 7085,
    "list_count": 9067,
    "descriptor_count": 3222,
    "token_count": 1405
}
account¶
Object Properties:
 
  • info (account_info) – account info
  • stats (account_stats) – account stats
JSON Example
{
    "info": {
        "account_id": "b3ebd840-1984-4bd3-88ca-89fe18ad5159",
        "email": "mroberts@montgomery.com",
        "organization_name": "{organization_name object}",
        "status": false
    },
    "stats": {
        "person_count": 3908,
        "list_count": 9023,
        "descriptor_count": 5303,
        "token_count": 786
    }
}
luna_list¶
Object Properties:
 
  • list_id (uuid4) – list id
  • account_id (uuid4) – account id
  • user_data (user_name) – user data about list
  • count_object_in_list (integer) – object’s count in list
  • last_update_time (iso8601) – time of last action with list (attach or detach face from list)
JSON Example
{
    "list_id": "620d1cc2-3626-4477-91b9-d606f9c17c26",
    "account_id": "40b351a9-330c-41af-8d2a-57cb025d2df6",
    "user_data": "sheila01",
    "count_object_in_list": 1251,
    "last_update_time": "1971-05-08T23:19:47"
}
luna_person¶
Object Properties:
 
  • person_id (uuid4) – person id
  • faces (_list_(uuid4)) – linked faces
  • create_time (iso8601) – time of face creating
  • user_data (user_name) – user data about face
  • account_id (uuid4) – account id
JSON Example
{
    "person_id": "a1645a28-59ba-425d-b34a-30c6ac10670f",
    "faces": [
        "53441489-be79-441d-8dac-fdfae85df3e7",
        "f4ceac07-cd1d-470b-9876-94675b4d6845",
        "66606c3d-75e3-4b66-ae7a-bed3e9a8ee84"
    ],
    "create_time": "1997-02-23T06:30:57",
    "user_data": "carpenternoah",
    "account_id": "0c541816-2802-4ca1-8a36-dfa4f132afe7"
}
auth_token¶
Object Properties:
 
  • token_id (uuid4) – token id
  • token_data (street_address) – token_data if auth is token else nothing
JSON Example
{
    "token_id": "ba8028c7-49e6-4cf0-a1a3-c278a17be78e",
    "token_data": "3383 Olson Island"
}
task_info¶
Object Properties:
 
  • duration (time) – duration of task (“in progress” if not ended)
  • target (_enum_(all)_(account)) – target of task
  • error_count (integer) – error count
  • progress (float) – task progress
JSON Example
{
    "duration": "21:19:18",
    "target": "all",
    "uuid4": "{target id object}",
    "error_count": 9937,
    "progress": 0.19442000007256866
}

taskGCOldDescriptorsDetails¶
Object Properties:
 
  • count_delete_descriptors (integer) – removed descriptors count
  • count_s3_errors (integer) – luna-image-store error count
JSON Example
{
    "count_delete_descriptors": 4064,
    "count_s3_errors": 8898
}
task_type¶
Object Properties:
 
  • task_type (_enum_(removing old descriptors)_(re-extract descriptors)) – “removing old descriptors” or “re-extract descriptors”
  • taskDetail (_enum_(taskGCOldDescriptorsDetails)_(taskReextractDetails)) – detail of task
JSON Example
{
    "task_type": "re-extract descriptors",
    "taskDetail": "taskGCOldDescriptorsDetails"
}
