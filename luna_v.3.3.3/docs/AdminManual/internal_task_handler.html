Internal task handler

post()[source]¶

Create internal task.

GET /task¶
Request Headers:
 
  • Authorization – basic authorization

Example request:

internal_task¶
Object Properties:
 
  • target (_enum_(all)_(uuid4)) – all or id of target
  • task_id (integer) – task id
  • subtask (integer) – subtask number
JSON Example
{
    "target": "uuid4",
    "task_id": 9975,
    "subtask": 2712
}

Example response:

HTTP/1.1 200 Ok
Vary: Accept
Content-Type: application/json

Output account will be represent in task_type

Message error is returned in format server_error.

Status Codes:
  • 200 OK – ok
  • 400 Bad Request – Bad query param
  • 404 Not Found – Task not found
  • 500 Internal Server Error – internal server error
