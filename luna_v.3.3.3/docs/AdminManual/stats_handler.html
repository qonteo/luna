get(series)[source]¶

Search element by id or email.

GET /realtime_statistics/{series}¶
Parameters:
  • series – extract_success|matching_success|errors
Query Parameters:
 
  • resource – resource to get statistics about. Will be ignored if not set. (“descriptors”, “search”, “match”, “identify”, “verify”)
  • error – luna API error code to get statistics about
  • aggregator – aggregation type (“max”, “min”, “mean”, “count”)
  • count_faces – result count faces on photo
  • limit – matching limit in match request
  • template – template in match request. 1 - person, 0 - descriptor
  • candidate – candidate in match request. 1 - dynamic list, 0 - static Luna API list
Request Headers:
 
  • Authorization – basic authorization

Example response:

HTTP/1.1 200 Ok
Vary: Accept
Content-Type: application/json
LUNA-Request-Id: 1516179740,d3abc2f6-70f1-4ae0-9d10-475103e0891d
stats result¶
Object Properties:
 
  • name (_enum_(extract_success)_(matching_success,errors)) – The name of current series
  • columns – Titles of cells in values’ batches
  • values – List of values’ batches
