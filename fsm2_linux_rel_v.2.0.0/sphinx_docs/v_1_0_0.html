v.1.0.0¶

Improvements:¶

  1. Now tests clear up database if proceed without errors.
  2. Added reports for clusterization and cross-matching tasks
  3. Now latex can be disabled in config (used for generating pdf-reports)
  4. Environment is checked on application startup
  5. LunaVL for pip installation in the ./external folder
  6. Now saving input images can be disabled in config
  7. REST documentation was translated into english
  8. Now we work with time in GMT timestamp millis format and store this time in Elasticsearch database, however all the user interfaces are processed in local rfc3339 time format
  9. Default tornado logger and custom application logger now both work in local time

Bugs fixed:¶

  1. Linker tasks are not updated two times in a row at start (caused an error before)

How to update from previous version:¶

You can only overwrite all the existing data by

python make_index.py

ATTENTION! ALL THE EXISTING DATA WILL BE LOST!
